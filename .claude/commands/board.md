# /board — Boardroom Perspectives Analysis

Run a boardroom perspectives analysis on the given topic.

Usage: /board [topic or decision]

Instructions:
1. Read `31-skills/boardroom-perspectives.md` for the full skill definition
2. Run the 5-perspective analysis on the given topic:
   - CEO / Strategic lens
   - CFO / Financial lens
   - CMO / Marketing lens
   - COO / Operational lens
   - Customer / End-user lens
3. Synthesize with weighted recommendations
4. Present findings clearly with actionable next steps

Saving:
- If the topic relates to Stellaris Ridge, append to `10-stellaris-ridge/_decisions.md`
- If the topic relates to Paw & Pantry, append to `20-paw-and-pantry/_decisions.md`
- Format: `## [Today's Date] — [Topic]` followed by the synthesis

Ask Carl to confirm before saving if the decision has significant implications.
