# /audit — Marketing Audit

Run a full 6-dimension marketing audit on a prospect.

Usage: /audit [url] [optional: client name]

Instructions:
1. Read `90-templates/marketing-audit.md` for the audit structure
2. Research the URL thoroughly — website, Google Business Profile, social media presence
3. Fill in all 6 dimensions with specific findings (not generic advice)
4. Score each dimension /10
5. Identify top 3 quick wins
6. Recommend specific Stellaris Ridge services based on gaps found

Save output to:
- `10-stellaris-ridge/audits/[client-name]-audit-[date].md`
- Note the file path in `10-stellaris-ridge/_active-projects.md`

If a client brief doesn't exist for this prospect, create one at:
`10-stellaris-ridge/clients/[client-name]/brief.md`
using the template at `90-templates/client-brief.md`

The audit should be specific enough that Carl or Jarod can walk into a sales call
and reference exact findings from the client's actual website.
