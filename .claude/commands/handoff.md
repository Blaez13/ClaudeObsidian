# /handoff — Session Handoff Generator

Generate a session handoff note for today's date.

Instructions:
1. Review what we accomplished in this session
2. Note anything in progress but not finished
3. List what the next session should start with (top 3 items)
4. Record any decisions made with their rationale
5. List files created or modified with brief description of changes
6. Note any open questions or blockers

Save the handoff to `01-daily/[today's date].md` under a `## Session Log` heading.
If the daily note already exists, append to it. If not, create it using the daily-note template.

Format: Use the session-handoff template from `90-templates/session-handoff.md`.

Also update:
- `10-stellaris-ridge/_active-projects.md` if any SR project status changed
- `10-stellaris-ridge/_decisions.md` if decisions were made
