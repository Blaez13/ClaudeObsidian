# Skill: Dog Food Safety

> Ingredient safety reference for all P&P recipe work.
> Check this before writing or approving any recipe.

## WHEN TO USE

- Before writing any recipe (Lyra)
- Before approving any recipe (Sage)
- When a customer asks about an ingredient

---

## TOXIC — NEVER USE

| Ingredient | Why Toxic |
|-----------|-----------|
| Onions, garlic, chives (raw, cooked, powdered) | Hemolytic anemia |
| Grapes and raisins | Kidney failure |
| Macadamia nuts | Neurological symptoms |
| Xylitol (artificial sweetener) | Hypoglycemia, liver failure |
| Chocolate, cocoa, caffeine | Theobromine toxicity |
| Alcohol | Toxic even in small amounts |
| Raw yeast dough | Ethanol production, bloat |
| Avocado (flesh, pit, skin, leaves) | Persin toxicity |
| Nutmeg | Neurological symptoms |
| Cooked bones (especially poultry) | Splintering hazard |

## CAUTION — USE SPARINGLY OR WITH NOTES

| Ingredient | Caution |
|-----------|---------|
| Salt | Sodium toxicity in large amounts — season minimally |
| Raw eggs | Biotin deficiency risk if fed regularly — cook for P&P recipes |
| Raw salmon/fish | Salmon poisoning risk — always cook |
| Dairy | Many dogs lactose intolerant — use sparingly, note in recipe |
| Nuts (except macadamia) | High fat, pancreatitis risk — limit amounts |

## SAFE STAPLES — ALWAYS APPROVED

| Ingredient | Benefits |
|-----------|---------|
| Chicken (cooked) | Lean protein |
| Turkey (cooked) | Lean protein |
| Beef (cooked) | Protein, iron |
| Brown rice | Digestible carbs |
| Sweet potato | Fiber, vitamins |
| Carrots | Beta-carotene, crunchy treat |
| Green beans | Low calorie, fiber |
| Blueberries | Antioxidants |
| Pumpkin (plain) | Digestive health |
| Eggs (cooked) | Complete protein |
| Oats (cooked) | Fiber |
